<h2>Research Interests</h2>
<p>My research interests include some sub-fields of <b>Computer Vision</b> and <b>Deep Learning</b>:</p>
<ul>
<li><p><b>Robust Deep Neural Networks</b>: Adversarial Example, Model Robustness and Stability, Adversarial Training</p>
</li>
</ul>
<ul>
<li><p><b>Interpretable Deep Learning</b>: Model Behavior via Generlaized Noise, Neuron Representation, Feature Visualization</p>
</li>
</ul>
<ul>
<li><p><b>Vision and Language</b>: Embodied Question Answering, Embodied Object Recognition</p>
</li>
</ul>
